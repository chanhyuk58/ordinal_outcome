Replication data for:
Testing the Compensatory Theory: A Survey Experiment on Covid-19 and Redistributive Preferences in the UK

Authors: Mariana Alvarado, Pablo Querubin, Kenneth Scheve, David Stasavage
2025

Overview: This folder contains replication data for Testing the Compensatory Theory: A Survey Experiment on Covid-19 and Redistributive Preferences in the UK. The following describes all included Dofiles and raw data files. All analyses were run using Stata/BE 18.0.

Description of Dofiles:
0_master: Contains program setups and executes dofiles 1 to 4.
1_data_cleaning: Uses rawdata.csv to clean the survey data.
2_main_analysis: Produces figure 1 and figure 2 as jpg and eps files.
3_appendix_analysis: Produces figures for Appendix in sections A, B, C.2, D, E, F, G
4_gpt_stats: Produces figures for Appendix in section C.1.2

Description of raw data files:
rawdata.csv contains the raw and original data from the Survey experiment.
classified_reas.dta, is a database that categorizes responses from the survey experiment. The classifications were generated using ChatGPT, version 4o-mini.
